Codex Gamer

O Codex Gamer é um site de guia sobre jogos, aqui terá muitas informações sobre os jogos que procura, como resumos, personagens, troféus etc. Aqui você poderá ler sobre os jogos para descobrir o que precisa ser feito ou saber se você realmente queira comprar o jogo. As informações serão atualizadas para que todos os jogos possam estar corretos nas informações dispostas para os consumidores, caso tenha algum jogo novo ou um jogo existente ganhe uma atualização que mude muitas coisas.
